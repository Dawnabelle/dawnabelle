Economic Value Atlas Lead Ember Developer

The Economic Value Atlas is an Ember application that is used as a decision-support tool. It enables a user to exhibit economic conditions among communities in the Portland-Vancouver region for display and analysis.

Version 2 of the Economic Value Atlas was a four month project with weekly deliverables. I performed accessibility audits and diagramed the flow of Version 1 for the developers and analysts. I redesigned portions of the application with UI and accessibility in mind. As a software developer, I wrote JavaScript within an Ember framework and worked with the ArcGIS for JavaScript API.

Version 2 of the Economic Value Atlas was released in February of 2020 to rave reviews from the stakeholders and positive feedback from users.
Visit the Economic Value Atlas

Regional Barometer GIS Developer & UI Designer

The Regional Barometer is an Esri Hub application that allows users to view easy-to-understand facts and figures with some accompanying narratives, access research and reports, and download data for analysis.

This project lasted nearly a year, with a team of two software developers, as well as other stakeholders within Oregon Metro. I joined in the final two months of the project executing weekly deliverables. I lent vital support in bringing the application up to WGAG 2.0 compliance as well as last minute design and content tweaks.

The release of the Regional Barometer was delayed due to COVID-19, but initial user tests were positive across the board. The application has been released as of April 2020 and is a helpful resource to the community and city planners and a template for other cities to build from.
Visit the Regional Barometer

Infographic Lead UI Designer

Infographic is a custom reports generator that accepts input from ArcGIS feature layers, as well as other sources, and outputs data into dynamic templates.

This project was an effort contributed to by the whole Dymaptic team. I designed the templates in Sketch and Balsamic before building the data import layout in JSON. I also designed the windows for users to upload and arrange their data and myriad template mockups for clients.

The application is slowly being released to select clients. Clients have reported success in presenting their data using the custom templates.
